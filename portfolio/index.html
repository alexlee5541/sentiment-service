The Architecture Evolution

A journey from a local script to a distributed cloud system.

Phase 1: The Prototype

Completed

Started as a local Python script using FastAPI and HuggingFace (FinBERT). Manual execution and local SQLite storage.

  • Single-threaded Python
  • Local processing
  • No isolation

Phase 2: Containerization

Completed

Dockerized the application stack. Split services into Microservices (Frontend, Backend, DB) using Docker Compose.

  • Docker Multi-stage builds
  • PostgreSQL persistence
  • Network isolation

Phase 3: Kubernetes & Zero-Trust

Live Production

Migrated to AWS EC2 (Jakarta) running K3s. Implemented Cloudflare Tunnel for "Ghost Mode" security (No open ports).

  • Orchestration: Kubernetes (K3s)
  • Security: Zero Inbound Ports
  • CI/CD: GitHub Actions Automated Pipeline
