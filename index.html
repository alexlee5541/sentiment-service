Featured Project: Real-Time Sentiment Analyzer

Infrastructure & Security (DevOps Showcase)

This project runs on a **Ghost Mode zero-trust architecture** on AWS EC2, eliminating all external attack surfaces. The Nginx, Streamlit, and PostgreSQL services are all containerized using Docker Compose.

DevOps & Networking

  • **Zero-Trust Ingress:** Cloudflare Tunnel (Ports 80/443 on AWS are closed)
  • **Routing:** Nginx routes root traffic to static files and subdomain traffic to Streamlit (Port 8501).
  • **Deployment:** Automated CI/CD via GitHub Actions and SSH.

Application Stack

  • **Frontend:** Streamlit (Interactive Visualization)
  • **Backend:** FastAPI (High-performance API)
  • **Database:** PostgreSQL (Data Persistence)
  • **Containers:** Docker Compose
